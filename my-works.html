NU Wait n Plate: Canteen Queueing System

An Android application designed for canteens, featuring random queue number generation, dine-in/take-out options, history tracking, and counters for ordered, no-show, and total sales. Includes call with sound, recall, and no-show functionalities.

Pawtify: Pet Care & Organizer App

An Android Studio application designed to help pet owners manage their beloved companions. Users can input pet details like name, age, weight, and birthday, set customizable reminders for feeding, walking, and appointments, and capture and store adorable 'pawtos' directly within the app.

SwiftBank: Secure Java Banking App

A console-based Java application for essential banking transactions like deposit, check balance, and withdraw, featuring PIN authentication and receipt generation for secure and transparent operations.

Hogwarts-Inspired Library System

A Java Application library management system with a touch of magic. Built for students and librarians, it allows users to log in, manage accounts, borrow and return books, and search for titles—even the rare, dusty ones.

Android Basic Calculator

Developed using Android Studio, this project showcases a fundamental understanding of mobile application development. It provides users with a clean and intuitive interface to perform standard arithmetic operations such as addition, subtraction, multiplication, and division. The calculator demonstrates core UI layout principles and event handling in Android, offering a practical example of a responsive and functional mobile utility.
