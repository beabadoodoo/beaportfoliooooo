Hogwarts-Inspired Library System

A Java library management system with a touch of magic.

Built for students and librarians, this project lets users log in, manage accounts, borrow and return books, and search for titles—even the rare, dusty ones hidden deep in the digital shelves. It keeps track of borrowing history and gives admins the tools to manage everything behind the scenes.

Clean design, smooth experience, and just enough Hogwarts energy to make library tasks feel a little less boring.
