Hogwarts-Inspired Library System

A Java library management system with a touch of magic.

Hogwarts Library System is a Java-based project built to manage a magical library experience, inspired by the world of witches and wizards.

It features secure student login, a full borrowing and return history, and a powerful search function—perfect for finding both standard textbooks and mysterious, ancient tomes. Students can easily borrow or return books, check current loans, and manage their profiles.
