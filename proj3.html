Android Basic Calculator

A simple, functional calculator developed with Android Studio.

This project is a straightforward, user-friendly basic calculator application built for Android devices using Android Studio. It demonstrates core Android UI development and event handling, providing essential arithmetic functions.

Users can perform addition, subtraction, multiplication, and division operations with ease. The app features a clean, intuitive interface, ensuring a smooth and responsive user experience for everyday calculations. It's a foundational project highlighting proficiency in mobile app development.
