SwiftBank: Secure Java Banking App

A console-based Java application for essential banking transactions.

SwiftBank is a robust console-based banking application developed in Java, designed to simulate fundamental banking operations. It prioritizes user security and convenience, allowing customers to perform essential transactions securely.

Key functionalities include **depositing funds**, **checking current balances**, and **withdrawing money**. Each transaction is secured with a **PIN authentication** system to ensure account safety. Upon completion of a transaction, the system automatically generates and displays a **receipt**, providing users with a clear record of their activity. This project demonstrates strong object-oriented programming principles and secure transaction handling in Java.
